ON-OFF KEYING — AMPLITUDE MODULATION
📡
OOK — ON/OFF KEYING
A subset of ASK · Amplitude Shift Keying
OOK encodes data by switching the carrier fully ON for bit 1 and fully OFF for bit 0. Nothing else changes — same frequency, same phase. It is the simplest possible form of radio communication. Your 433 MHz aircon remote, gate opener, and wireless doorbell all use OOK. Trivially captured and replayed by any SDR including HackRF.
OOK WAVEFORM AMPLITUDE DOMAIN
Carrier switches between full power and zero. Frequency stays fixed the entire time. No sweep, no phase shift. One frequency, two amplitude states.
OOK SPECTRUMFREQ DOMAIN
Single narrow spike at carrier frequency. High peak, zero spread. Immediately visible on HackRF waterfall. Easy to detect, easy to jam.
WHY IT MATTERSSECURITY
Zero replay resistance. Capture the burst once with HackRF, replay it to your own gate or sensor. This is why Meshtastic and LoRa chose CSS instead of OOK.

FREQUENCY SHIFT KEYING — FREQUENCY MODULATION
🎚️
FSK — FREQUENCY SHIFT KEYING
Discrete Frequency Tones · Constant Amplitude
FSK encodes data by switching between two or more defined frequencies. Amplitude stays constant throughout. Bit 1 transmits at f1, bit 0 transmits at f2. On a waterfall you see two horizontal lines the signal bounces between. Used in POCSAG paging at 152 MHz, APRS amateur radio at 144 MHz, some TPMS sensors, and dial-up modems. Better noise resistance than OOK because amplitude variations do not corrupt data.
FSK WAVEFORM FREQUENCY DOMAIN
Amplitude is constant — the wave height never changes. Only the frequency jumps between f1 and f2 per bit. The transitions between frequencies carry the data. No amplitude changes at all.
FSK SPECTRUMFREQ DOMAIN
Two spikes visible — one per tone. Signal energy alternates between the two frequencies as data changes. Classic FSK signature. POCSAG and APRS both appear this way on HackRF.
WHY IT MATTERSUSE CASES
POCSAG paging at 152 MHz uses FSK — hospital pagers broadcast patient data in plaintext, decodable with HackRF. APRS at 144.390 MHz broadcasts GPS positions of vehicles and weather stations.

PHASE SHIFT KEYING — PHASE MODULATION
🔄
PSK — PHASE SHIFT KEYING
Phase Angle Shifts · Constant Amplitude and Frequency
PSK encodes data by shifting the phase angle of the carrier. Amplitude stays constant. Frequency stays constant. Only where the wave starts in its cycle changes. BPSK uses two phase states — 0° and 180°. QPSK uses four. Harder to detect passively because spectrum looks identical for both 0 and 1. Used in GPS, ACARS VDL2 at 136 MHz, and WiFi base modulation.
PSK WAVEFORM PHASE DOMAIN
Watch for the phase flip at bit transitions — the wave inverts suddenly at boundaries. Amplitude and frequency are identical for both 0 and 1. Only the phase angle changes.
PSK SPECTRUMFREQ DOMAIN
Single spike at carrier — looks identical to OOK spectrum. Phase information is invisible on a simple spectrum analyzer. You need an IQ constellation diagram to decode PSK data.
WHY IT MATTERSUSE CASES
GPS uses BPSK at 1575.42 MHz. ACARS VDL2 uses QPSK at 136 MHz — aircraft operational messages. More bandwidth efficient than OOK or FSK. QPSK encodes 2 bits per symbol using 4 phase states.

CHIRP SPREAD SPECTRUM — ITS OWN CATEGORY
WHY CSS IS FUNDAMENTALLY DIFFERENT FROM OOK, FSK, AND PSK
OOK changes amplitude. FSK changes frequency between fixed tones. PSK changes phase angle. CSS does none of these.

CSS continuously sweeps frequency across the entire bandwidth — never staying at any fixed frequency. Data is encoded in the starting offset of each sweep. This makes CSS immune to narrowband interference, very hard to jam, and extremely difficult to replay with a basic SDR.
📶
CSS — CHIRP SPREAD SPECTRUM
Continuous Frequency Sweep · Spread Spectrum · LoRa · Meshtastic
Each symbol is a chirp — a continuous frequency sweep across the full bandwidth. The spreading factor SF7 to SF12 controls how many chips per symbol. Higher SF = more spread = more noise resistance = slower data rate. Energy is spread across 125 to 500 kHz bandwidth. Appears as a raised noise floor to narrowband receivers. Your HackRF can see the band but decoding requires knowing the exact SF, BW, and CR parameters.
CSS CHIRP WAVEFORM CONTINUOUS SWEEP
Each symbol is a continuous frequency sweep — never fixed at any single frequency. The starting offset of the sweep encodes the symbol value. Arrows show chirp direction per bit symbol.
CSS SPECTRUMFREQ DOMAIN
Wide flat spectral density across full bandwidth. No sharp spike. Appears as slightly raised noise floor. Invisible to narrowband receivers. Adjust the SF slider to see processing gain change.
WHY IT MATTERSSECURITY + RANGE
Replay attacks fail because timing precision requirements exceed basic SDR capture capability. Meshtastic adds message ID deduplication on top. Two independent layers block naive replay — physical and protocol.
